科技
======

+ 概述
    + 科技是基于主公的，体现在整个游戏生命环节
    + 设计成类似lol的，然后提高不满足感，和变化性
    + 攻击
        + 部队攻击力+1%（4）
        + 武将武力+1
        + 同阵营的部队的攻击力+1%
        + 对面同阵营的部队攻击-1%
        + 宝石带来的攻击力+1% (4)
        + 击杀对手之后下次攻击必定暴击
        + 击杀对手之后士兵数恢复2%
        + 主将阵营的武将部队攻击力+1%
        + 部队暴击+1%攻击力（4）


    + 防御
        + 部队防御力+1%(4)
        + 武将统帅+1

    + 通用
        + 部队士兵数+1%（4）
        + 武将智力+1
        + 获得金钱+1
